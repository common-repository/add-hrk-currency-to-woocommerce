== Description ==

Adds Croatian Kuna (HRK) currency in Woocommerce. Also puts currency to the right of the price. When activated, HRK appears in WooCommerce currency list. Works with new WooCommerce 2.0+.

== Installation ==

1. Upload `Add_HRK_Currency_in_WooCommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now in WooCommerce currency list there is HRK
